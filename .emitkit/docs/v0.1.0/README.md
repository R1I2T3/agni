# Agni Notification Engine API SDK Documentation

Version: 0.1.0
API Version: 1.0.0

## Description

Agni is a self-hostable, lightweight, and scalable notification engine. It provides REST endpoints for sending notifications across multiple channels (email, SMS, push, webhook), managing in-app notifications, handling WebPush subscriptions, and administering applications.


## API Servers

- http://localhost:8080
- http://localhost:4000
